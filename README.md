# Pokes1

Aplicación web desarrollada con **Next.js (App Router)** que consume la **PokéAPI** para mostrar un catálogo de Pokémon con rutas dinámicas.

Cada Pokémon se puede seleccionar desde un menú y se muestra su información junto con su sprite en estilo pixel art.

---

## Tecnologías

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- PokéAPI

---

## Rutas

| Ruta | Descripción |
|------|-------------|
| `/` | Página principal |
| `/pokemon` | Catálogo de Pokémon |
| `/pokemon/pikachu` | Vista individual |
| `/pokemon/eevee` | Vista individual |

---

## API utilizada

PokéAPI  
https://pokeapi.co/

Sprites obtenidos desde:  
https://raw.githubusercontent.com/PokeAPI/sprites

---

##  Ejecutar el proyecto en local

```bash
npm install
npm run dev
 ```
Abrir en el navegador:
http://localhost:3000/pokemon
